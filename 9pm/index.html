Angular Directives and Pipes Examples

1. ngIf Directive

Welcome, user!

<div *ngIf="isLoggedIn">Welcome, user!</div>

2. ngFor Directive

  • {{ product.name }} - {{ product.price | currency }}
<ul>
  <li *ngFor="let product of products">{{ product.name }} - {{ product.price | currency }}</li>
</ul>

3. ngClass Directive

<button [ngClass]="{'active': isActive, 'disabled': !isActive}">Click Me</button>

4. ngStyle Directive

This text has dynamic styling.
<div [ngStyle]="{'color': color, 'font-size': fontSize}">This text has dynamic styling.</div>

5. ngModel Directive

Hello, {{ userName }}!

<input [(ngModel)]="userName" placeholder="Enter your name" />

6. ngSwitch Directive

Admin Dashboard
User Dashboard
Guest Dashboard
<div [ngSwitch]="userRole">
  <div *ngSwitchCase="'admin'">Admin Dashboard</div>
  <div *ngSwitchCase="'user'">User Dashboard</div>
  <div *ngSwitchDefault>Guest Dashboard</div>
</div>

7. ngContainer Directive

Welcome back, user!

 
<ng-container *ngIf="isLoggedIn">Welcome back, user!</ng-container>

8. ngTemplateOutlet Directive

This is a reusable template!

 
<ng-template #template>
  <p>This is a reusable template!</p>
</ng-template>
<div *ngTemplateOutlet="template"></div>

9. ngComponentOutlet Directive

<ng-container *ngComponentOutlet="dynamicComponent"></ng-container>

10. formControl & formControlName Directive

<form [formGroup]="myForm">
  <input formControlName="username" placeholder="Username" />
  <input formControlName="password" type="password" placeholder="Password" />
</form>

11. ngPlural Directive

No messages One message {{ messagesCount }} messages 
<div [ngPlural]="messagesCount">
  <ng-template ngPluralCase="=0">No messages</ng-template>
  <ng-template ngPluralCase="=1">One message</ng-template>
  <ng-template ngPluralCase="other">{{ messagesCount }} messages</ng-template>
</div>

12. ngNonBindable Directive

{{ user.name }}

<div ngNonBindable>
  <p>{{ user.name }}</p> 
</div>

13. ngContent Directive

Card Content
 
<app-card>
  <div>Card Content</div>
</app-card>

14. ngModelGroup Directive

<form #form="ngForm">
  <div ngModelGroup="address">
    <input [(ngModel)]="address.street" name="street" />
    <input [(ngModel)]="address.city" name="city" />
  </div>
</form>

15. async Pipe

{{ userData$ | async }}
<div>{{ userData$ | async }}</div>

16. currency Pipe

{{ product.price | currency:'USD' }}
<div>{{ product.price | currency:'USD' }}</div>

17. date Pipe

{{ currentDate | date:'short' }}
<div>{{ currentDate | date:'short' }}</div>

18. json Pipe

{{ user | json }}
<pre>{{ user | json }}</pre>
